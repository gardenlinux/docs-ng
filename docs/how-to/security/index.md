# Security Configuration Guides

Configure security features to harden your Garden Linux deployment.

## Security Guides

- [SSH Hardening](/how-to/security/ssh-hardening) - Configure SSH for secure access
- [Secure Boot](/how-to/security/secure-boot) - Configure and verify secure boot
- [Time Configuration](/how-to/security/time-configuration) - Configure NTP and time synchronization for security

## Why Security Configuration Matters

Garden Linux is designed with security in mind, but proper configuration is essential:

- **SSH hardening** prevents unauthorized access
- **Secure boot** ensures system integrity
- **Time synchronization** is critical for security protocols and logging

## Security Principles

Garden Linux follows these security principles:

- **Immutable images** - Reduces attack surface
- **Minimal footprint** - Only essential packages included
- **Regular updates** - Security patches applied promptly
- **Compliance ready** - CIS benchmark compatible
