# API Reference

Technical reference for Garden Linux APIs and command-line tools.

## Available Documentation

- [Python Library API](/reference/api/python-lib) - Python gardenlinux library API reference
- [CLI Reference](/reference/api/cli) - Command-line interface reference

## Python Library

The `python-gardenlinux-lib` provides programmatic access to Garden Linux functionality:

- Image management
- Build system integration
- Testing utilities
- Configuration management

See the [Python Library API](/reference/api/python-lib) for complete documentation.

## Command-Line Interface

Garden Linux provides CLI tools for:

- Building images
- Managing configurations
- Running tests
- Deploying images

See the [CLI Reference](/reference/api/cli) for command documentation.
