# Release Information

Information about Garden Linux releases, maintenance windows, and changelogs.

## Release Documentation

- [Maintained Releases](/reference/releases/maintained-releases) - Currently maintained releases with support dates
- [Release Notes](/reference/releases/release-notes) - Release notes and changelog

## Release Information

Garden Linux follows a predictable release cadence:

- **Regular releases** - New versions released on schedule
- **Maintenance windows** - Clear support timeframes
- **Security updates** - Prompt security patches

For the philosophy behind releases, see [Release Cadence](/explanation/release-cadence).

## Finding the Right Release

### Latest Stable

For production use, choose the latest stable release from [Maintained Releases](/reference/releases/maintained-releases).

### Long-Term Support

Some releases have extended support windows. Check the maintenance dates in the release table.

### Development Builds

Nightly builds are available for testing but should not be used in production.
